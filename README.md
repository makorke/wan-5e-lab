# wan-5e-lab

本仓库用于归档和发布多个独立的 HTML 页面，涵盖不同主题与用途的静态网页内容。所有页面均为独立文件，不依赖特定后端服务，可直接在浏览器中打开或部署于任意静态托管平台。

## 项目简介

wan-5e-lab 是一个以 HTML 页面归档为主的仓库，旨在整理、保存并展示各类静态页面作品。每个页面均以自包含的方式编写，便于快速查看、分享或二次修改。仓库内容不针对特定网站或域名，仅作为通用页面集合使用。

## 目录说明

```
wan-5e-lab/
├── pages/           # 存放所有独立 HTML 页面
├── assets/          # 页面所需的公共资源（样式、脚本、图片等）
└── README.md        # 本文件
```

- **pages/**：每个 HTML 文件对应一个独立页面，命名清晰，方便识别内容。
- **assets/**：若多个页面共用同一资源，可统一放置于此目录下，避免重复。

## 页面归档说明

- 所有页面均为纯静态 HTML，无外部依赖（除公共资源外）。
- 页面内容可能包含演示、工具、实验或示例性功能，不保证长期可用或更新。
- 每个页面均可在本地直接打开运行，无需配置服务器环境。
- 部分页面可能需要现代浏览器支持（如 ES6+、CSS Grid 等）。

## 维护说明

- 本仓库不定期更新，新增或修改页面会通过 commit 记录说明。
- 欢迎提交 Issue 反馈页面问题或建议，但不保证及时处理。
- 不接受直接向仓库提交代码或页面内容的 Pull Request，除非事先沟通。
- 仓库维护者保留对页面内容进行修改、删除或归档的权利。

## 使用方式

1. 克隆或下载本仓库。
2. 打开 `pages/` 目录下的任意 HTML 文件即可查看页面。
3. 如需部署，可将整个仓库上传至任意静态托管服务（如 GitHub Pages、Netlify 等），并设置 `pages/` 为根目录。

## 许可

本仓库内容无特定许可声明，所有页面仅供学习、参考或个人使用。请勿将页面内容用于商业或违法用途。
